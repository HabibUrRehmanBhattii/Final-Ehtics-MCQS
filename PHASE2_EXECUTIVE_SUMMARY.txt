## How to Monitor Your Performance

### Quick Check (Console)
```javascript
// Run anytime in your browser console
MCQApp.getWebVitals()

// Returns your current Core Web Vitals:
{
  lcp: 1245,    // Largest Contentful Paint (ms)
  inp: 85,      // Interaction to Next Paint (ms)
  cls: 0.0042,  // Cumulative Layout Shift (unitless)
  fcp: 920,     // First Contentful Paint (ms)
  ttfb: 245     // Time to First Byte (ms)
}
```

### Production Monitoring
Vitals are **automatically tracked** in your analytics:
- Check DevTools → Network → filter for `/api/analytics/track`
- Look for events with `"type": "metric"`
- Vitals sent every 5 seconds

### Build a Dashboard
Query your analytics data with SQL (documented in [PHASE2_RUM_GUIDE.md](./PHASE2_RUM_GUIDE.md)):
```sql
SELECT metric_name, AVG(value) as avg_value
FROM page_metrics
WHERE ts > datetime('now', '-7 days')
GROUP BY metric_name;
```
